Background - Shorthand Property

To shorten the code, it is also possible to specify all the background properties in one single property. This is called shorthand property.

            The shorthand property for background is background.

            When using the shorthand property the order of property values are: 

            
                
    
                    
  • background-color
    • 
                    
  • background-image
    • 
                    
  • background-repeat
    • 
                    
  • background-attachment
    • 
                    
  • background-position
    • 
                

            

            It does not matter if one of the property values are missing, as long as other ones are in this order.
