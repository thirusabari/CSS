Text Color

The color property is used to set the color of the text.
        
        With CSS, a color is must often specified by: 

        
            
    
                
  • a color name - like red
    • 
                
  • a HEX value - like #ffBB00
    • 
                
  • an RGB Value - like rgb(255,0,0)
    • 
            

        
        The default text color of a page is defined in the body selector.
