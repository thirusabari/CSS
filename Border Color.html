Border Color

            The border-color property is used to set the color of the four borders.

            The color can be set by: 
            
    
                
  • name - specify a color name, like "red".
    • 
                
  • Hex - specify a hex value, like "#ff0000"
    • 
                
  • RGB - specify a RGB value, like "rgb(255,0,0)"
    • 
                
  • transparent
    • 
            

            The border-color property can have from one to four values (for the top border, right border, bottom border, left border).

            If the border-color is not set, it inhents the color of the element.
