HexaDecimal Colors

RGB Values can also be specified using the HexaDecimal
        color values in the form: 
        #RRGGBB. When RR(red), GG(green), BB(blue) are hexadecimal values between 00 and FF (same as decimal 0-255).
        For Example, #FF0000 is displayed as red, because red is set to highest value(FF) and the others are set to the lowest value(00). 
        Note:
        HEX values  are case insensitive: 
        #ff0000 is the same as #FF0000
